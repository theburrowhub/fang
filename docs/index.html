What you get

Everything a terminal deserves

Built with Rust for maximum performance. No Electron, no Node — just a single static binary.

⚡
Blazing fast
Written in Rust with zero-allocation hot paths. Async preview loading keeps the UI at 60 fps regardless of file size.
🎨
Syntax highlighting
Powered by syntect with the base16-ocean dark theme. 300+ language grammars, fully offline.
🔍
Fuzzy search
Real-time fuzzy filtering via SkimMatcherV2. Press / and results appear as you type.
⚙️
Makefile integration
Automatically detects Makefiles, parses all targets, and lets you run make <target> with a single keypress. Output streamed live in the preview panel.
📐
Responsive layout
Three-panel view (sidebar · file list · preview) that adapts to your terminal width. Collapse panels on the fly with s and p.
🦀
Single binary
No runtime dependencies. Drop fang into your $PATH and it works everywhere.

Get started

Install in seconds

Choose your preferred method.

🍺  Homebrew
brew tap theburrowhub/tap
brew install fang
📦  Cargo
cargo install --git https://github.com/theburrowhub/fang
⬇️  Binary download
# macOS (Apple Silicon)
curl -L https://github.com/theburrowhub/fang/releases/latest/download/fang_VERSION_aarch64-apple-darwin.tar.gz | tar -xz
sudo mv fang /usr/local/bin/
🛠  Build from source
git clone https://github.com/theburrowhub/fang
cd fang
make install
