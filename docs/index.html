What you get

Everything a terminal deserves

Single binary. No Electron. No Node. Just Rust.

⚡
Blazing fast
Written in Rust with zero-allocation hot paths. Async preview loading keeps the UI at 60 fps regardless of file size.
🎨
Syntax highlighting
Powered by syntect with base16-ocean.dark. 300+ language grammars, fully offline.
🔍
Fuzzy search
Real-time fuzzy filtering via SkimMatcherV2. Press / and results appear as you type.
/
⎇
Git operations
status, fetch, pull, push, log, stash, diff and more — all from a menu. Output streams live into the preview panel.
g
⚙️
Makefile integration
Detects Makefiles, parses targets, runs make <target> with live streaming output.
m
💻
Shell commands
Run any command (with aliases) inside fang — output in preview. Or open in a new terminal split (zellij, tmux, kitty, iTerm2…).
: ;
📋
Create files from clipboard
Create a new empty file or paste from clipboard — text, images copied from the web, binary files. Works with macOS and Linux.
n N
🚀
Open with default app
Open any file or directory with the system's default application. Finder on macOS, xdg-open on Linux.
o
📐
Responsive layout
3-panel view (sidebar · file list · preview) that adapts to terminal width. Toggle panels with s and p.
📊
Header bar
Top bar shows current Git branch and detected dev environments: Python venv, Go, Node.js, Rust — detected asynchronously.
🖥️
Window title
Terminal window title updates to the current path on every navigation via OSC escape sequences. Works in any xterm-compatible terminal.
🦀
Single binary
Drop fang into your $PATH and it works everywhere.

Get started

Install in seconds

🍺  Homebrew
brew tap theburrowhub/tap
brew install fang
📦  Cargo
cargo install --git https://github.com/theburrowhub/fang
⬇️  Binary (macOS, Apple Silicon)
curl -L https://github.com/theburrowhub/fang/releases/latest/download/fang_VERSION_aarch64-apple-darwin.tar.gz | tar -xz
sudo mv fang /usr/local/bin/
🛠  Build from source
git clone https://github.com/theburrowhub/fang
cd fang
cargo build --release
./target/release/fang

Terminal split support — ; opens commands in a new pane. Requires one of: Zellij, tmux, kitty (allow_remote_control yes), WezTerm, Ghostty, iTerm2, or any Linux terminal emulator.
